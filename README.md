# 🚀 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website built to showcase my skills, projects, and experience as a Front-End Developer.

## ✨ Overview
This project represents my ability to design and develop production-ready front-end applications using modern web technologies.

## ⚡ Key Features
- Fully responsive design (mobile, tablet, desktop)
- Smooth animations using Framer Motion
- Contact form integration using EmailJS
- Clean and reusable React components

## 🛠 Tech Stack
- React.js
- JavaScript
- Tailwind CSS
- Framer Motion
- EmailJS

## ⚙️ Tools
- Git & GitHub
- Vercel
